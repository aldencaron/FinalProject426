Console

dhcp-v000-0-00XYZ:~ $

Console Description

Use this console to make SQL queries to the database as you normally would. Make sure to enter one-line statements. If you want to clear the console, type 'clear' on a single line. Be sure to have the mouse focus on the console before typing. If you want to insert into your own database, simply go to AJAXTesterPHP.php and change the mysqli connection. Note, you can only change the connection when using the console. The GET and POST interfaces will require the change of all mysqli connections from the SettlersOfCarolina_orm.php.
